# Contacts By Call History

## What's this all about

A simple Flutter application that can retrieve your contacts in Android and sort them by call history, so you know which contacts are lying dormant in your phone, and which ones you want to delete.

### Instructions for using the app

To build the app from source -

1. Install VS Code
2. Install the Flutter extension for VS Code
3. Clone this repository, and open it in VS Code
4. Run `flutter build apk --release`
5. Manually install the APK on your device

To install the pre-built APK -

1. Download the latest version from the releases pages of this repository
